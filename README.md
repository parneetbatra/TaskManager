# TaskManager

A simple task management application built with a .NET backend and Angular frontend. The app supports creating, viewing, editing, deleting, and filtering tasks, with clear task status tracking and timestamp metadata.

## Functional requirements

The user must be able to:

- View a list of tasks
- Create a new task
- Edit an existing task
- Delete a task
- Mark a task as complete or incomplete
- Filter tasks by status: all, active, completed

Each task should include at minimum:

- title
- description
- status
- created date
- updated date
